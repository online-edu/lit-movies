# \<lit-movies>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i lit-movies
```

## Usage
```html
<script type="module">
  import 'lit-movies/lit-movies.js';
</script>

<lit-movies></lit-movies>
```

## Testing using karma (if applied by author)
